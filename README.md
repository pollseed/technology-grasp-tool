technology-grasp-tool: can see grasp of technology

## Required

* Java 1.7.0.5
* Gradle 2.1
* eclipse

## Setting

```
$ gradle build
$ gradle eclipse
```

## Started

### Run

```
$ java -jar build/libs/technology-grasp-tool.jar server ./config/server.yml
```

### Test

```
# GET Access for parameters
localhost:8080/hello-world

# Admin Operational MENU
```
